图片批量转换为WebP格式工具

🎯 功能说明

这个工具可以批量转换当前文件夹下的所有图片为WebP格式，支持JPG、PNG、BMP、GIF、TIFF等常见格式。

📦 包含文件

  • convert_to_webp.py - Python脚本（推荐）
  • convert_to_webp.bat - Windows批处理文件
  • convert_to_webp.ps1 - PowerShell脚本

🚀 使用方法

方法一：Python脚本（推荐）

**优点：**无需额外安装软件，质量最高

  1. 自动安装依赖并运行：

    双击运行 convert_to_webp.bat

  2. 手动运行：

    # 安装依赖
pip install Pillow

# 基本转换（保留原文件）
python convert_to_webp.py

# 指定质量（1-100）
python convert_to_webp.py -q 90

# 无损压缩
python convert_to_webp.py -l

# 删除原文件
python convert_to_webp.py --remove-original

# 指定目录
python convert_to_webp.py -d "D:\我的图片"

方法二：PowerShell脚本

**优点：**Windows原生支持

**前提条件：**需要安装FFmpeg

# 使用winget安装FFmpeg
winget install FFmpeg

# 或使用Chocolatey
choco install ffmpeg

使用方法：

# 基本转换
.\convert_to_webp.ps1

# 指定质量
.\convert_to_webp.ps1 -Quality 90

# 无损压缩
.\convert_to_webp.ps1 -Lossless

# 删除原文件
.\convert_to_webp.ps1 -KeepOriginal:$false

# 指定目录
.\convert_to_webp.ps1 -InputDir "D:\我的图片"

⚙️ 参数说明

Python脚本参数

  • -d, --directory : 目标目录（默认：当前目录）
  • -q, --quality : 压缩质量 1-100（默认：85）
  • -l, --lossless : 使用无损压缩
  • --remove-original : 删除原始文件

PowerShell脚本参数

  • InputDir : 输入目录（默认：当前目录）
  • Quality : 压缩质量 1-100（默认：85）
  • Lossless : 使用无损压缩
  • KeepOriginal : 保留原文件（默认：true）

📊 支持格式

输入格式： JPG, JPEG, PNG, BMP, GIF, TIFF, TIF
输出格式： WebP

💡 使用建议

  1. 质量设置：

    • 85-95: 高质量，适合重要图片
    • 70-85: 平衡质量与文件大小
    • 50-70: 较小文件，适合网页使用

  2. 无损 vs 有损：

    • 无损：保持原始质量，文件较大
    • 有损：较小文件，适合大多数用途

  3. 建议操作流程：

    • 先备份重要图片
    • 使用默认设置测试转换效果
    • 根据需要调整质量参数

❗ 注意事项

  • 转换前请备份重要图片
  • WebP格式在某些老版本软件中可能不被支持
  • 转换过程中请勿关闭程序窗口
